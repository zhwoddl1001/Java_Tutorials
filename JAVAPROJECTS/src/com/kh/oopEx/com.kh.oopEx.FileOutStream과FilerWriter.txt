FileOutputStream 과 FileWriter.txt


특징               FileOutputStream                 FileWriter
데이터 유형        바이너리 데이터                  문자데이터

주 사용 대상       이미지 오디오 비디오 텍스트      텍스트 파일
                   파일 등 모든 데이터

데이터 처리 단위   바이트 단위                      문자 단위

인코딩 지원        기본적으로 인코딩 처리 없음     UTF-8등 문자 인코딩 지원
            *인코딩 ? -> 컴퓨터 언어를 사람이 읽을 수 있도록 변환

사용 용이성       텍스트 처리 복잡                  텍스트 처리에 적합
버전              Java 1.0                          Java 1.0

요즘에는 (JDK 7 버전 이상) 부터는 Files 많이  사용
Files 이미지 비디오 텍스트 등 단순히 처리 가능
